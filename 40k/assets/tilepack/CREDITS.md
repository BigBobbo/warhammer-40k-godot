# Tilepack sprite credits

Top-down tank, terrain-decoration and effect sprites in this directory are
from Kenney's asset packs (https://kenney.nl), matching the "Top-down Tanks
Redux" naming. Kenney game assets are released under Creative Commons Zero
(CC0) — public domain, free for commercial use, no attribution required
(credit given here anyway, thanks Kenney!).

Source drop: the repository-level TILEPACKS/ folder; only the sprites the
game actually uses are imported here.
