# 40kdc Dataset Attribution

The JSON files in this directory are extracted from the
[`@alpaca-software/40kdc-data`](https://www.npmjs.com/package/@alpaca-software/40kdc-data)
npm package (version 1.0.19), the dataset behind [40kdc](https://40kdc.alpacasoft.dev).

- **Enrichment data** (ability DSL, phase mappings, scoring encodings, etc.):
  © Alpaca Software and the 40kdc community contributors, licensed
  [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)
  (<https://github.com/wn-mitch/40kdc-data>).
- **JSON Schemas**: CC0.
- Stat lines and points values are numerical facts. Ability and rules text are
  not reproductions of Games Workshop text — abilities are community-authored
  structured mechanics (the Ability DSL).

**Public deployment requirement** (from the package license): any publicly
accessible application or service that ships this data as part of its
end-user product must display a visible credit containing the text
**"Powered by 40kdc-data"** and a link to <https://40kdc.alpacasoft.dev> in a
user-accessible location (footer, about page, or credits section).

These files are regenerated by `scripts/40kdc/extract.mjs` — do not hand-edit;
re-run the extraction to update to a newer dataset version.
